# StudyAlpha-Ai-Agent — ULTRA

**Shubham Mahajan · AI Agent Arc**

StudyAlpha is an autonomous multi-agent study coach that generates personalized study schedules, creates grounded quizzes, evaluates answers, predicts weak topics using a small ML model, and adapts future revision plans — implemented as a modular Python package with a Streamlit demo and reproducible Kaggle notebook.
